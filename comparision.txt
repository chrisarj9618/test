|              | **Fargate (Normal)**                                                                                                               | **Fargate Spot**                                                                                                                                                 |
| ------------ | ---------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Pros**     | - Guaranteed availability<br>- No interruptions mid-run<br>- Predictable performance & timing<br>- Simple setup, no retries needed | - **Much cheaper** (up to \~70% off)<br>- Same performance as Normal when running<br>- Ideal for short-lived, retryable jobs<br>- Can mix with Normal for safety |
| **Cons**     | - **More expensive** (pay full rate)<br>- Might be overkill for ephemeral CI jobs<br>- Less cost efficiency for short bursts       | - Can be interrupted with **2 min notice**<br>- Capacity not guaranteed<br>- Requires retry logic in CI<br>- Slightly more setup to handle interruptions         |
| **Best for** | Critical, time-sensitive jobs that must run without fail and without delay                                                         | Ephemeral, fault-tolerant jobs (like most CI/CD stages) that can retry or resume quickly                                                                         |
